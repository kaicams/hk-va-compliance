**香港虚拟资产与金融服务合规专家**

一个专注于香港虚拟资产（VA）及金融服务监管合规的 Claude Skill。涵盖香港证监会（SFC）及香港金管局（HKMA）相关监管体系，包括 VATP 发牌制度、AML/CFT、Travel Rule、KYT、稳定币、Staking、DeFi 及 RWA 等领域。

---

## 三大功能模式

### 模式一：专业咨询（Professional Consultation）

向 Skill 提出任何香港虚拟资产相关监管问题，即可获得结构化合规分析及法规依据。

**输出格式：**

* **Summary（总结）**：监管结论
* **Detailed Analysis（详细分析）**：具体监管要求拆解
* **Regulatory Reference（法规依据）**：引用法规、指引及段落
* **Actionable Next Steps（下一步建议）**：优先处理事项

**覆盖主题：**

| 主题                    | 主要法规来源                                               |
| --------------------- | ---------------------------------------------------- |
| VATP 发牌制度             | AMLO Cap.615、VATP Licensing Handbook、VATP Guidelines |
| AML / KYC / CDD / EDD | AML Guideline                                        |
| Travel Rule           | AMLO、AML Guideline Chapter 12                        |
| KYT / 链上监控            | SFC Layering Circular                                |
| OTC 虚拟资产交易            | AMLO Cap.615                                         |
| 稳定币                   | HKMA Stablecoin Regime                               |
| Staking               | SFC Staking Circular                                 |
| DeFi                  | SFC 相关监管指引                                           |
| RWA                   | SFC 产品扩展通函                                           |

---

### 模式二：合规文件Gap 分析（Policy Gap Analysis）

上传公司政策、程序或操作手册，对照香港监管要求进行差距分析。

**适用文件：**

* Compliance Manual
* AML Policy
* KYC Policy
* Transcation Monitoring Policy
* Travel Rule Policy
* KYA/KYT Procedure
* Internal Control Manual
* Operational Guideline

**输出内容：**

1. 文件识别及适用监管范围
2. Gap Analysis 分析表
3. 合规成熟度评分（0-100）
4. 总体评级
5. 整改路线图


### 模式三：文件修订（Policy Edit & Rewrite）

自动修订政策文件、补充缺失章节，并生成符合香港监管要求的版本。

**适用场景：**

* 重写 Compliance Policy
* 补充章节
* 更新程序
* 完善治理架构要求
* 增加制裁筛查流程

## 法规来源

所有法规文件均按需从公开资料实时获取，包括：

* AMLO Cap.615
* SFO Cap.571
* VATP Licensing Handbook
* VATP Guidelines
* AML Guideline
* SFC Circulars
* HKMA Stablecoin Regime

如无法取得原始文件，系统将明确说明，而不会仅凭记忆生成答案。
